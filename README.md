# 🛒 Java HashMap - Order Management

Aplicação em Java que organiza pedidos por status utilizando `HashMap` e `List`, com foco em prática de lógica e estruturas de dados.

---

## 🚀 Objetivo

- Praticar Programação Orientada a Objetos (POO)
- Utilizar `List` (ArrayList)
- Trabalhar com `HashMap`
- Agrupar dados por categoria (status)
- Percorrer e exibir dados organizados

---

## 🛠 Tecnologias

- Java 11
- Eclipse IDE
- Git & GitHub

---

## 📂 Estrutura

- `model` → Classe `Pedido`
- `main` → Classe executável `TestePedido`

---

## ⚙️ Funcionamento

- Criação de pedidos com cliente, status,  valor, numeroPedido
- Armazenamento em uma lista
- Agrupamento por status (`PENDENTE`, `ENVIADO`, `ENTREGUE`)
- Impressão organizada no console

---
